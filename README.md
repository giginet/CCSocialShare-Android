# CCSocialShare-Android

This is the sample project of [CCSocialShare](https://github.com/giginet/CCSocialShare) for Android.

## Build

Set following Environment variables

- ANDROID_SDK_ROOT
- NDK_ROOT
- ANT_ROOT
- COCOS_X_ROOT

```sh
$ cocos build -p android --android-studio
```

## Requirements

- cocos2d-x (3.11.1)
- Latest Android SDK
- Latest Android NDK
